ThumbFish

A simple utility for Chemists that integrates with Windows and provides molecule Thumbnail, Preview, Search and other useful features.

View project onGitHub

About ThumbFish

ThumbFish is a set of Windows Shell Extension handlers developed to help people working with the following chemical structure files:

  • MOL (MDL Molfile Format - V2000 and V3000)
  • RXN (Reaction file - V2000 and V3000) SMILES (Simplified Molecular Input Line Entry Specification)
  • SMARTS (SMiles ARbitrary Target Specification)
  • SDF (Structure Data File - V2000 and V3000)
  • RDF (Reaction Data File - V2000 and V3000)
  • CML (Chemical Markup Language)

The handlers makes it easy to identify and search these chemical files without opening the associated application. These handlers are NOT dependent on any specific commercial Chemistry application. It uses the open source chemistry organic toolkit named Indigo by GGA Software. ThumbFish contains the following Shell Extension Handlers for Windows OS:

  • Thumbnail Handler
  • Preview Handler
  • IFilter Search Handler
  • Shell Context-Menu Handler
